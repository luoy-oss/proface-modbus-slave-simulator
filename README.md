# Modbus TCP从机调试软件

## 项目简介

这是一个功能完整的Modbus TCP从机调试软件，主要用于工业自动化领域的Modbus协议调试和测试。软件可以作为Modbus TCP从机，接收来自PLC、HMI、SCADA系统等Modbus主机的读写请求，并提供实时的通信监控和数据管理功能。

## 主要功能

### 1. Modbus TCP服务器功能
- 支持标准Modbus TCP协议
- 可配置IP地址和端口（默认502）
- 支持Unit ID过滤（0表示忽略Unit ID检查）
- 支持IPv4地址绑定（0.0.0.0监听所有接口）

### 2. 数据存储与模拟
- **线圈状态**：65536个布尔值存储
- **离散输入**：65536个布尔值存储  
- **输入寄存器**：65536个16位整数值存储
- **保持寄存器**：65536个16位整数值存储
- 预置测试数据，方便快速测试

### 3. 协议支持
- **读操作**：
  - 功能码01：读线圈
  - 功能码02：读离散输入
  - 功能码03：读保持寄存器
  - 功能码04：读输入寄存器
- **写操作**：
  - 功能码05：写单个线圈
  - 功能码06：写单个寄存器
  - 功能码15：写多个线圈
  - 功能码16：写多个寄存器

### 4. 配置选项
- **地址映射模式**：
  - 标准Modbus模式：地址从0开始
  - Pro-face模式：地址从1开始
- **字节序设置**：
  - 大端模式（高位低字）
  - 小端模式（低位高字）
- **功能码选择**：可选择性启用/禁用特定功能码

### 5. 数据监控界面
- **通信日志**：实时显示Modbus请求和响应
- **线圈状态监控**：显示地址、状态和值
- **离散输入监控**：显示地址、状态和值
- **输入寄存器监控**：显示地址和值
- **保持寄存器监控**：显示地址和值
- 支持双击修改数据值

### 6. 数据模拟功能
- **自增任务**：寄存器值自动递增
- **位翻转任务**：线圈状态自动翻转
- **时间数据**：自动生成当前时间数据
- **日期数据**：自动生成当前日期数据
- 可配置模拟间隔和范围

### 7. 性能优化
- 使用array.array代替Python列表，减少95%内存占用
- Treeview缓存机制，优化GUI响应速度
- 消息队列频率限制，避免GUI阻塞
- 增量更新，只更新变化的数据项

## 运行环境

### Python环境运行
1. 确保安装Python 3.6或更高版本
2. 运行命令：`python modbus_slave_debugger.py`

1. 直接运行`dist/ModbusSlaveDebugger.exe`

## 配置说明

### 配置文件
软件会自动生成`modbus_config.json`文件保存配置：
- 服务器配置（IP、端口、Unit ID）
- 功能码启用状态
- 地址映射和字节序设置

### 默认配置
- IP地址：192.168.1.113
- 端口：502
- Unit ID：1
- 地址映射：标准Modbus（地址从0开始）
- 字节序：大端模式

## 注意事项

1. **网络配置**：确保防火墙允许Modbus端口（默认502）通信
2. **IP地址**：使用0.0.0.0可监听所有网络接口
3. **性能考虑**：大量数据模拟可能影响GUI响应
4. **数据安全**：软件仅用于测试环境，不建议用于生产环境
